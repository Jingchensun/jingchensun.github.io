Chapter 4: Sampling Distributions

Motivating Example: Coin Flipping

Flip a coin \(n = 10\) times. Each flip is a Bernoulli trial:

\[ X \sim \text{Bern}(p), \quad X = \begin{cases} 1 & \text{(head with prob } p) \\ 0 & \text{(tail with prob } 1 - p) \end{cases} \]

  • \( \mathbb{E}[X] = p \)
  • \( \text{Var}[X] = p(1 - p) \)

Let the sample be \(X_1, X_2, \ldots, X_{10}\), then:

\[ T = X_1 + X_2 + \cdots + X_{10} \sim \text{Binom}(10, p) \]

Standard error: \( SE[T] = \sqrt{\text{Var}[T]} = \sqrt{np(1 - p)} = 2.5 \)

Simulation for Approximate Sampling Distribution

  1. Randomly draw 1 sample of size \(n\)
  2. Compute number of heads \(T^{(1)}\)
  3. Repeat \(N\) times to get \(\hat{T} = \{T^{(1)}, \ldots, T^{(N)}\}\)
  4. Approximate PMF: \( \hat{p}(x) = \frac{\text{count}(\hat{T} = x)}{N} \)

Example: Exponential Distribution

Let \(X \sim \text{Exp}(\lambda)\) with \(\lambda = 1/15\)

  • \(\mathbb{E}[X] = \mu = 1/\lambda = 15\)
  • \(\text{Var}[X] = \sigma^2 = 1/\lambda^2 = 225\)

For sample mean \(\bar{X}\) of size \(n = 100\):

  • \( \mathbb{E}[\bar{X}] = \mu = 15 \)
  • \( \text{Var}[\bar{X}] = \sigma^2 / n = 2.25 \)
  • \( SE[\bar{X}] = \sqrt{\text{Var}[\bar{X}]} = 1.5 \)

Although \(\bar{X}\) follows a Gamma distribution, we use simulation to approximate its distribution and confirm theoretical properties.

Central Limit Theorem (CLT)

For large \(n\), the distribution of the sample mean \(\bar{X}\) is approximately normal:

\[ \bar{X}_n \approx \mathcal{N}\left(\mu, \frac{\sigma^2}{n}\right) \]

\[ \lim_{n \to \infty} P\left(\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \leq z\right) = \Phi(z) \]

So for sufficiently large \(n\), we can use normal approximation for the sampling distribution of the sample mean.
