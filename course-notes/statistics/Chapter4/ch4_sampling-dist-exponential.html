example: sampling distribution for exponential distribution

n <- 100
lambda <- 1/15
#xplot = seq(0,40,length=101) uncomment for plot of the pdf for the exp dist
#plot(xplot,dexp(xplot,rate=lambda),type="l",col="green",ylab="pdf")

Generate a sample of size n from Exp(lambda)

x_sample <- rexp(n, rate=lambda)  
xbar <- mean(x_sample)
head(x_sample)
## [1] 12.941313  1.772523 11.200354 16.011609 13.870416 11.085784

sample mean xbar = 14.6339043

Create a sampling distribution for the mean of the sample

N <- 10**3  # number of samples 
T_hat <- numeric(N) # create a vector to hold statistic T=xbar for each sample
# loop to create sample, calculate xbar, store it in That array
for (i in seq(1,N)) {   
  x_sample <- rexp(n, rate=lambda)  
  xbar <- mean(x_sample)
  T_hat[i] <- xbar
}   
head(T_hat) # T_hat is our approx to sampling distribution for xbar)
## [1] 11.97783 17.12764 13.23496 13.78525 18.69144 14.11843

Histogram of appoximate sampling dist

hist(T_hat,main="") # histogram of approx sampling dist

comparison of mean and std error

# theory (see notes)
mu = 1/lambda
sigma = 1/lambda
mean_theory <- mu
stderr_theory <- sigma/sqrt(n)
# simulation
mean_approx <- mean(T_hat)
stderr_approx <- sd(T_hat)
n=100, N=1000 mean std dev
theory 15 1.5
approx 15.0413941 1.5696494

Note: as N increases, the approximation approaches the theoretical (exact) results.

For large sample size n, sampling distribution of the mean is approximated by a normal distribution

qqnorm(T_hat)  # qqplot of sampling dist
qt <- seq(-3,3,length=101)
lines(qt,mean_theory+stderr_theory*qt,col='red') # line with mean and std dev of data

The qqplot shows the approximate sampling distribution T_hat for the mean of n=100 trials from exponential dist. Red line corresponds to normal distribution with same mean and stderr as the theoretical result.

The agreement means T_hat is approximated well by a normal distribution. This is because the Central Limit Theorem says T_hat approaches a normal dist as n->infinity, and here n=100 is sufficiently large that T_hat is approximated by a normal distribution.
